# Teeopticals Website UI Kit

Modernized marketing site for Teeopticals. A click-through prototype showing:
- Landing hero with bilingual copy + iris halos
- Services split (Clinic + Retail)
- Booking form with reason chips
- Real bilingual patient reviews
- Clinic info + location

## Files
- `index.html` — entry, loads Babel + components
- `Nav.jsx` — sticky top nav
- `Hero.jsx` — landing hero with breathing logo
- `Services.jsx` — Clinic + Retail service grids
- `Booking.jsx` — book-an-eye-test panel
- `Reviews.jsx` — reviews carousel (real bilingual content)
- `Footer.jsx` — clinic info

Reviews load from `../../reviews.json`.
